# JiangShu

> Building AI-powered workflows, study systems, and tools that are actually useful.

I like turning messy inputs into structured systems —
videos into notes, schedules into workflows, and learning into something reusable.

## What I'm building
- **AI workflows** for note-taking, extraction, and automation
- **Study systems** that make learning easier to organize and review
- **Developer tools** with a focus on clarity, speed, and usefulness
- **Knowledge pipelines** that turn daily inputs into long-term assets

## Featured projects
- **[videonote-ai](https://github.com/Health-525/videonote-ai)**
  AI-powered workflows that turn videos into structured learning notes.

- **[timetable](https://github.com/Health-525/timetable)**
  A lightweight timetable + study automation system with sync workflows.
